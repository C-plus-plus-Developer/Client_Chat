Клиент для чат-сервера
Клиентское приложение для подключения к многопоточному чат-серверу. Поддерживает все функции сервера: регистрацию, авторизацию, обмен сообщениями и административные операции.

Основные возможности
Для пользователей:
Подключение к серверу по IP-адресу

Регистрация нового аккаунта

Авторизация в системе

Отправка приватных сообщений

Отправка публичных сообщений

Просмотр истории сообщений

Просмотр списка пользователей

Для администраторов:
Авторизация в административной панели

Просмотр всех пользователей

Просмотр онлайн-пользователей

Блокировка/разблокировка пользователей

Удаление пользователей

Просмотр заблокированных пользователей

Требования
C++17 компилятор

CMake 3.10+

Linux/Unix система (используются POSIX сокеты)

Запущенный сервер чата на порту 7777

Сборка и запуск
Сборка:

mkdir build && cd build
cmake .. && make
Запуск:

./Client_Chat
По умолчанию клиент подключается к 127.0.0.1:7777

Структура проекта
Основные файлы:
main.cpp - точка входа в программу

Chat.h / Chat.cpp - основной класс клиента:

ChatClient - управление соединением и взаимодействием с сервером

connectToServer() - подключение к серверу

sendCommand() - отправка команд на сервер

run() - главный цикл клиента

Класс ChatClient:
Соединение: TCP-сокет с таймаутами на операции

Отправка команд: гарантированная доставка полных команд

Получение ответов: буферизация и обработка разорванных пакетов

Интерфейс: консольное меню с разделами для пользователей и администраторов

Особенности реализации
Сетевое взаимодействие:
TCP-сокеты с надежной доставкой

Таймауты на операции чтения (3 секунды)

Буферизация для обработки разорванных пакетов

Гарантированная отправка полных команд

Безопасность:
Валидация ввода на стороне клиента

Обработка сетевых ошибок

Корректное завершение соединения

Защита от зависаний через select()

Пользовательский интерфейс:
Консольное меню с интуитивной навигацией

Разделение прав пользователей и администраторов

Чтение полных строк с пробелами

Форматированный вывод ответов сервера

Административные аккаунты
По умолчанию в серверной части настроены:

admin / admin123

moder / moder123

Для входа в админ-панель используйте команду "5" в главном меню.

Ограничения
Длина сообщения: до 4096 байт

Кодировка: предполагается UTF-8

Таймаут соединения: 3 секунды на операцию чтения

Однопоточность: синхронные операции

Отладка и устранение проблем
Проверка соединения:
Убедитесь, что сервер запущен на порту 7777

Проверьте доступность порта: telnet 127.0.0.1 7777

Убедитесь, что нет блокировки фаерволом

Типичные ошибки:
"Connection failed" - сервер не запущен или недоступен

"Send failed" - разрыв соединения во время передачи

Пустой ответ - таймаут ожидания ответа

Некорректные данные - несовпадение версий протокола

Интеграция с сервером
Клиент совместим с серверной частью чата, ссылка будет прикреплена ниже.
